# Llamia Project

## Architecture Overview

### Core Components
1. **Planner** - Breaks down tasks into executable steps
2. **Coder** - Generates code implementations
3. **Executor** - Runs and validates code
4. **Critic** - Analyzes results and suggests improvements

### Workflow
User Request → Planning → Code Generation → Execution → Validation → Final Output

### Key Technologies
- Python 3.10+
- LLM Integration (via API)
- Automated Testing Framework
- Research and Retrieval (RAG) System
